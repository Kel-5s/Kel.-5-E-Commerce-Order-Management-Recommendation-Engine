Fungsi main() merupakan pusat pengendali sistem pada program E-Commerce Order Management & Recommendation Engine.
Modul ini bertugas menginisialisasi seluruh struktur data yang digunakan, seperti Queue untuk antrian pesanan berdasarkan
tier pelanggan, Stack untuk menyimpan riwayat transaksi, BST untuk katalog produk, serta Graph untuk
sistem rekomendasi produk. Selain itu, program juga menggunakan List, Set, dan Dictionary untuk menyimpan
data order selesai, order yang dibatalkan, dan histori pelanggan.

Pada awal program, sistem akan membuat data produk otomatis menggunakan fungsi generate_produk(100)
lalu memasukkannya ke dalam BST katalog. Setelah itu, program menampilkan tampilan awal CLI (Command Line Interface)
dan menunggu input dari pengguna. Sistem CLI memungkinkan pengguna menjalankan berbagai perintah seperti membuat order,
mencari produk, memproses antrian, hingga melihat laporan transaksi melalui terminal.

Program berjalan menggunakan perulangan while True, sehingga sistem akan terus aktif sampai pengguna keluar dari program.
Input pengguna dibaca menggunakan input(), kemudian diproses dengan teknik parsing command, yaitu memisahkan
perintah utama dan parameter tambahan. Dengan konsep ini, modul main() berfungsi sebagai penghubung seluruh fitur dan struktur data agar
sistem e-commerce dapat berjalan secara terintegrasi dan terorganisir.
